
# Bone Tride

A simple topdown shooter game in C++ SFML.

## Features

- Player Health Bar
- Simple Score System
- Infinte Ammo (was not botherd adding ammo)
- Zombies Randomly Spawn
- Sound FX
- Enemy Health
- Player Upgrades
